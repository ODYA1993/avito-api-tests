# Тест-кейсы для API объявлений (Задание 2.1: Тесты API)

**Базовый URL:** `https://qa-internship.avito.com`  
**Версия API:** `/api/1/` и `/api/2/`

## Список тестируемых методов

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/1/item` | Создание объявления |
| GET | `/api/1/item/{id}` | Получение объявления по ID |
| GET | `/api/1/{sellerID}/item` | Получить все объявления по идентификатору продавца |
| GET | `/api/1/statistic/{id}` | Получить статистику по айтем id |
| DELETE | `/api/2/item/{id}` | Удаление объявления |

## ТЕСТ-ДИЗАЙН (ТЕХНИКИ)

### Граничные значения
| Поле | Граница | Значение |
|------|---------|----------|
| price | Мин. положительное | 1 |
| price | Ноль | 0 |
| price | Отрицательное | -1 |
| name | Мин. длина | 1 символ |
| name | Пустая строка | "" |
| name | Макс. длина (если есть) | 255 / 1000 символов |
| sellerID | Мин. положительное | 1 |
| sellerID | Ноль | 0 |

### Классы эквивалентности
**Поле sellerID:**
- ✅ Валидные: положительные целые числа (1, 100, 999999)
- ❌ Невалидные: 0, отрицательные, строки, null, пустота

**Поле name:**
- ✅ Валидные: латиница, кириллица, цифры, пробелы
- ❌ Невалидные: пустая строка, null, только пробелы

**Поле price:**
- ✅ Валидные: положительные целые числа
- ❌ Невалидные: 0, отрицательные, дробные, строки

## ИДЕМПОТЕНТНОСТЬ
| Метод | Идемпотентен? | Пояснение |
